# Contractor License Feed — ARCHIVED

This niche was killed 2026-07-18 after saturation analysis.

## Why Killed
- LimeLeads (9/mo), Apollo, Salesgenie, Shovels.ai already sell enriched contractor data with verified emails/phones
- Our dataset was inferior: no emails (CSLB legally prohibited), no phones (TX has none)
- No competitive wedge — a buyer comparing us to LimeLeads loses every time
- 'Newly licensed' recruiting angle had real demand but our data didn't support it reliably

## What Worked
- TX TDLR open data API (34K records, easy to scrape)
- Gumroad API for product creation
- GitHub Pages for landing page + dashboard
- Landing page + member dashboard built and deployed

- dashboard.html (password-protected member dashboard)
- scripts/refresh_wa_hvac.py, scripts/refresh_tx.py
- data/*.csv (WA + TX contractor records)
- OUTREACH_MESSAGES.md (10 drafted, never posted)
